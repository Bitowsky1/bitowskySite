title===About gooFLA

GooFLA is a program made for World of Goo, that converts in-game cutscenes called "movies" into Adobe Animate .fla projects and viceversa! It was made for the users that don't want to use the <a href="http://goofans.com/developers/world-of-goo-movie-editor">World of Goo Movie Editor (Goovie)</a>

section===Requirements

In order to fully use the program, you will need the following programs:
- gooFLA (obviously, duh)
- Adobe Animate*
- GooTool (Check the <a href="https://discord.gg/6BEecnD">Goofans Discord</a> for that)
- Goovie
- World of Goo game

*gooFLA was tested on the 2024 version, but that might not be necesarly the required minimum. It has also been tested on Adobe Flash so it should be mostly compatible with that as well.
